# MedVirtual CTA Button — Style Specification

**Primary CTA:** `REQUEST AN INTERVIEW`  
**Alternate CTA:** `MEET YOUR NEXT HIRE`

Target scale on 1080×1350 ad: button height **124px** · text **56px** · horizontal padding **58px**.

---

## PRIMARY — #1 Yellow physical button

| Property | Value |
|---|---|
| Font family | Be Vietnam (`Be Vietnam`) |
| Font weight | 800 (ExtraBold) |
| Font size | 56px |
| Button height | 124px |
| Button width | ~778px (scales to label) |
| Corner radius | 62px (full pill ends) |
| Face | `#F2D72E` |
| Text | `#073F2A` |
| Outline | `#075C37` · 10px |
| Extrusion | 14 hard steps · `#052818` |
| Top highlight | Cream `#FFF4D6` bar along upper edge |
| Arrow stroke | 9px · length ~34px |
| Padding | 58px left · arrow zone ~80px right |
